## Usage

> If you used the inference code provided in this repository for each model, you will get the scores automatically. Otherwise, you can follow the instructions below to evaluate your results.

### 1. Prepare Your Result Videos

Organize the videos generated by your models according to the following directory structure.
```text
/path/to/your/pred_videos/
├── {video_id}__{seed}.mp4
├── {video_id}__{seed}.mp4
└── ... (more video files following the naming pattern)
```

```text
/path/to/your/gt_videos/
├── {video_id}__{seed}.mp4
├── {video_id}__{seed}.mp4
└── ... (more video files following the naming pattern)
```

### 2. Run Score Computation

```bash
python evaluate.py \
  --pred_dir /path/to/your/pred_videos \
  --gt_dir /path/to/your/gt_videos
```

This script will print the metrics (PSNR, SSIM, LPIPS) for each video.
